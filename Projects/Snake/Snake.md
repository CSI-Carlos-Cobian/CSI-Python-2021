## Code Reading Practice

### Comment Code:
1. Create `Snake.py`
2. Copy the first block of code.
3. Run the code.
4. Comment each line of code.
5. Create a commit.
6. Read the next block of code. 
7. Include any new lines of code in your file.
8. Comment and repeat 4-7
